# Bankist
Bankist is a simple two-player dice game simulator.

## How to play
- Clone the repository and open the index.html page to start the game


## Technologies
-  CSS
-  HTML
-  JavaScript

## Key Concepts
-  Event Handling
-  DOM Manipulation
